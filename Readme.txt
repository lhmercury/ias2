El proyecto ias2 es una plantilla web multipropósito basada en Bootstrap llamada "Flexor". Se trata de un repositorio que contiene los archivos de una página web estática con estructura de carpetas para activos, formularios y videos. El proyecto incluye múltiples páginas HTML como índice, misión, cotización y política de cookies. Es un proyecto de código abierto sin actividad colaborativa significativa en este momento.

Puntos Funcionales Principales
Plantilla web responsive basada en Bootstrap
Múltiples páginas HTML (inicio, misión, cotización, política de cookies)
Estructura organizada con carpetas para activos, formularios y videos
Archivo de registro de cambios para documentación
